Para ejecutar el "backend":
En la consola de windows o cualquier otra terminal, estando en la carpeta server:
    - npm install
    - node (o nodemon) index.js

Para ejecutar el "frontend":
En la consola de windows o cualquier otra terminal, estando en la carpeta client:
    - npm install
    - npm start
